타입 변환
	+ 연산에서의 문자열 자동 타입 변환
	자바에서 + 연산자는 두 가지 기능을 가진다.
	피연산자가 모두 숫자일 경우에는 덧셈 연산을 수행.
	피연산자 중 하나가 문자열일 경우에는 나머지 피연산자도 문자열로 자동 변환되어 문자열 결합 연산을 수행한다.
	
	숫자 + 숫자 = 덧셈연산 = 숫자
	"문자열" + 숫자 = "문자열" + "숫자" = 결합 연산 = "문자열숫자"
	숫자 + "문자열" = "숫자" + "문자열" = 결합 연산 = "숫자문자열"
	
	특정 부분을 우선 연산하고 싶을 경우, 해당 부분을 괄호()로 감싸준다. 괄호는 최우선으로 연산을 수행한다.
